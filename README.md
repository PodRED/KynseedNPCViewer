A save file inspector for [Kynseed](https://kynseed.com/) by Pixelcount Studios.

Pulls information for all currently living NPCs from your save file.

Save files can be found in your Kynseed directory in the "Saves" folder (or "Experimental Saves" if you are on the Experimental branch)

Access live version of this project here : <https://podred.github.io/KynseedNPCViewer/>
